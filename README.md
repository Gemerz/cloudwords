## 1.start with options

```js
var options = {
  id:'cloudWords',
  word:["Dear God","give me","a break"],
  auto:true,
  height:300
}

var cloudWords = new cloudWords(options);
```

## 2. configure

The following object sets up the default settings of cloudWords.

```js
defaults = {
  id: 'cloudWords',
  word: ['Gemer', 'zhang'],
  color: ['#e1f3f4', '#eaf1f1'],
  width: win.innerWidth,
  height: '200',
  position: 'center',
  fontsize: '100',
  auto: false,
  index: 0,
  second: 4,
}
```
